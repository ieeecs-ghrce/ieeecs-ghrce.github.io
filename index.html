What is Kotlin Language?

  • Kotlin is a new programming language from JetBrains.
  • It first appeared in 2011 when JetBrains unveiled their project named “Kotlin”
    • Kotlin is an Open-Source Language.
    • Kotlin is the Official Language of Android.
  • Kotlin also supports procedural programming with the use of functions
  • “Statically typed programming language”.

Benefits of Kotlin Language

  • Kotlin compiles to JVM bytecode or JavaScript 
  • Kotlin programs can use all existing Java Frameworks and Libraries 
  • Kotlin can be learned easily
  • Automatic conversion of Java to Kotlin
  • Code reviews are not a problem
  • Kotlin’s null-safety is great

NullPointerExceptions

      val   name: String = null   // tries to assign null, won’t  compile.
  
      fun   getName() : String = null   // tries to return null, won’t  compile.

Features of Kotlin Language

  • Versatile.
    • Use Kotlin for any types of Enterprise java EE development.
    • 100% compatible with all JVM frameworks.
    • Write code in Kotlin and target js to run on node.js or browser.
  • Kotlin avoids the null pointer exception.
  • No performance impact. Very small run time.
  • Lean Syntax

Consider the following example -

public class Address {

   private String city;
   private Country country;

   public Address(String city, Country country) {
       this.city = city;
       this.country = country;
   }

   public String getCity() {
       return city;
   }
   public void setCity(String city) {
       this.city = city;
   }

   public Country getCountry() {
       return country;
   }
   public void setCountry(Country country) {
       this.country = country;
   }
}
